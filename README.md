# HR-Analytics-Dashboard-Power-BI-Repository
Overview : This project visualizes HR data across job titles, gender, age, qualifications, salaries, and leave balances.
Built to demonstrate data-driven insights for workforce planning and HR decision-making.
Dataset size: 50 employees across multiple roles (Analysts, Developers, Managers, Leads, HR).
Key Insights : 1) Balanced workforce: Gender split is exactly 50% male and 50% female.
               2)Salary distribution: Ranges from ₹500K (HR Executive) to ₹1.42M (Product Analytics Lead).
               3)Age diversity: Workforce spans bins from 20–40 years, evenly distributed across genders.
	       4)Hiring trend: Significant onboarding spikes in Jul 2025, Oct 2025, and Jan 2026, showing rapid growth phases.

Features : 1) Headcount analysis by job title, gender, and age.
           2) Salary benchmarking with min, max, and average per role.
	   3) Qualification impact on salary distribution (Bachelor’s, Diploma, Master’s).
	   4) Cumulative headcount growth by joining date.
